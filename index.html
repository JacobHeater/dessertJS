Multiple Application Example

An example showing how to embed multiple applications in a single dessertJS parent application. This example shows how you can abstract core functionality into their own applications. This would be useful in situations where you would want to reuse entire applications. Modules are already a great way to abstract functionality, but modules are coupled to their applications. When you want to reuse functionality that isn't tied to a specific application, it would be better to abstract that logic into a separate application.

A good example of this is seeing how I have created a parent app that hosts child apps, like the calculator.
