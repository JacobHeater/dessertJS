Event Driven Module Example

This module offers three events that we can leverage to auto save the contents of the textboxes. We can use the view.nameChanged event to track when the name textbox value changes. We can also use the view.ageChanged event to track when the age textbox value changes. Finally, we can use the view.doneEditing event to track when the user is finished editing the textboxes, where I have created a simple mock REST call to save the data.
